# Vault Web App 🌐

The main dashboard for Vault. A Progressive Web App (PWA) built with React and Vite, serving as the primary interface for managing your secure credentials, security cycles, and recovery settings.

## 🚀 Features

- **PWA**: Installable on Desktop/Mobile as a native app.
- **Biometric Ready**: Integrated with the `@vault/shared` biometric utility for quick unlock.
- **Master Management**: Full control over your vault, including recovery key generation and password rotation.
- **Theme**: Premium Dark mode with sleek glassmorphism aesthetics.

## 🛠️ Development

Run from the root of the project:
```bash
pnpm dev
```
Local access: `http://localhost:5173`

## 📦 Deployment

The web app is deployed to **Firebase Hosting**.
- **Live URL**: https://vault.darshannaik.com

Deployment is automated via GitHub Actions on merge to `main`.
